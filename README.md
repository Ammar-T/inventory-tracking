# Shopify Challenge - Fall 2022 

## Tech Used

#### Backend - Node, Express, GraphQL, MongoDB

#### Frontend - React, Material

## View on Replit
https://inventory-tracking.ammar-t.repl.co/

## Run Locally
1. Create static build from frontend.
```
cd client
npm install
npm run build
```
2. Run server.
```
cd ../server
npm install
npm run start
```
